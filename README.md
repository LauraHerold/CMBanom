## CMBanom

Notebooks and scripts to reproduce results in 
"Nearly Full-Sky Low-Multipole CMB Temperature Anisotropy: III. CMB Anomalies" by L. Herold, G. E. Addison, C. L. Bennett, H. C. Nofi, J. L. Weiland ([arXiv:2509.03720](https://arxiv.org/abs/2509.03720)).

Foreground-cleaned maps (ELC maps) are available on Hayley Nofi's [github]([https://github.com](https://github.com/hnofi/ELC_Paper1)) ([paper](https://arxiv.org/abs/2509.03718)); Planck maps are taken from the [Planck Legacy Archive](https://pla.esac.esa.int/#home); WMAP maps ar taken from [LAMBDA](https://lambda.gsfc.nasa.gov).

Parts of this code are inspired by Jessica Muir's [github](https://github.com/jessmuir/cmbanomcov_muir-adhikari-huterer) ([paper](https://arxiv.org/abs/1806.02354)); this also makes use of the [polymv](https://github.com/oliveirara/polymv) package by Renan Alves de Oliveira ([paper](https://arxiv.org/abs/1812.02654)).
