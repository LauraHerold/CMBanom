## CMBanom

Notebooks and scripts to reproduce results in "Nearly Full-Sky Low-Multipole CMB Temperature Anisotropy: III. CMB Anomalies" by L. Herold, G. E. Addison, C. L. Bennett, H. C. Nofi, J. L. Weiland ([arXiv:2509.03720](https://arxiv.org/abs/2509.03720)).

Parts of this code are inspired by Jessica Muir's [github](https://github.com/jessmuir/cmbanomcov_muir-adhikari-huterer) ([paper](https://arxiv.org/abs/1806.02354));  
this also makes use of the [polymv](https://github.com/oliveirara/polymv) package by Renan Alves de Oliveira ([paper](https://arxiv.org/abs/1812.02654)).
