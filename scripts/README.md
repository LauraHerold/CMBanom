Main script to compute CMB anomaly statistics (CMBanom.py): large parts are adapted from Jessica Muir's [github](https://github.com/jessmuir/cmbanomcov_muir-adhikari-huterer) and [paper](https://arxiv.org/abs/1806.02354). 
This also makes use of the [polymv](https://github.com/oliveirara/polymv) package by Renan Alves de Oliveira ([paper](https://arxiv.org/abs/1812.02654)). 

Scripts to create the simulated maps (1), compute power spectra and correlation function (2) and the anomaly statistics for sims (3), real maps (4) and fake-cleaned maps (5).
